# rtcp-go

支持
* 内网穿透
* SSH动态绑定转发

# [Download](https://github.com/zhengoogle/rtcp-go/releases)


# Usage
* [内网穿透](#内网穿透)
* [SSH动态绑定转发](#SSH动态绑定转发)

### 内网穿透
```
⇒  ./rtcp -r 127.0.0.1:8080 -l 127.0.0.1:11020

```

### SSH动态绑定转发
```
⇒  ./rtcp -r 127.0.0.1:8080 -l 127.0.0.1:11020

```
